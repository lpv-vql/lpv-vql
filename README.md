暇なときは自作言語を考えたりしています。

好きな言語の特徴

- 動的型付け
- 括弧を多用しない(endとかで結ぶ)
- 記号をあまり使わない
- 暗黙の型変換をしない
- コードがインデントに左右されない
- 式指向

誤解しないでほしいのですが、**好き**と**得意**は別物ですよ?
